# BJJ Analyzer - AI Sports Analytics Platform

A cross-platform mobile application for analyzing Brazilian Jiu-Jitsu matches using AI.

## Stack
- **Mobile**: React Native + Expo
- **Backend**: Python Flask + MediaPipe + OpenCV
- **AI**: Pose Estimation, Person Detection, Position Classification
